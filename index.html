KMNIST Character Classification

Texas A&M University

Abstract

This project examines handwritten Kuzushiji character recognition using the KMNIST dataset. We compare logistic regression with several CNN architectures including a Simple CNN, ResNet-like CNN, and an Improved CNN with BatchNorm and Dropout. A tuned CNN reaches 97% accuracy. We also extend the project to few-shot learning using Siamese networks achieving 77% one-shot accuracy.

Methodology

Dataset

KMNIST contains 70,000 28×28 grayscale images— 60,000 for training and 10,000 for testing. Images are normalized and split into 48,000 training and 12,000 validation samples.

Models

  • Logistic Regression: 60% accuracy.
  • Simple CNN: 93% accuracy.
  • ResNet-like CNN: 94% accuracy.
  • Improved CNN: 96.7% accuracy.
  • TunedImprovedCNN: 97.0% accuracy.

Hyperparameter Tuning

Grid search over learning rate, dropout, batch size. Best config: LR=0.001, Dropout=0.4, Batch=128.

Conclusion

CNNs dramatically outperform linear models for character recognition on KMNIST. Hyperparameter tuning yields a robust 97% model, and metric learning techniques provide strong few-shot performance. Future work includes data augmentation and deeper architectures.
